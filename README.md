
# Covid-19 Vax Locations



## Summary

A place where you can see current cases, people that had been vaccinated and sign up to make an appointment to get vaccinated near your current location


## Tools Used

* JavaScript
* Node.js
* Express
* React
* Context API
* Redux
* Stripe
* Apollo Server
* Graphql
* Mongodb
* Mongoose
* 3rd Party API
* Heroku
